Challenges Overcome

IIS Deployment Complexity

Encountered technical difficulties with wfastcgi implementation for Django deployment on IIS server due to deprecated technology and lack of active maintenance support.

Solution: Researched alternative approaches and migrated to httphandler, successfully resolving deployment obstacles. Configured DNS routing and troubleshooted Django-IIS integration, resulting in successful production deployment.

Data Persistence Bug

Critical bug where students who previously registered for exams in a different series encountered data persistence issues when registering for a new series, with old subject selections incorrectly carrying over.

Solution: Conducted comprehensive analysis of database schema and API fetching code architecture. Restructured primary keys (PK) and foreign keys (FK) of database entities, and refactored API fetching logic to properly clear previous registration data.

Production Deployment Timing

Lack of system maintenance notification to end users created unnecessary confusion and system instability when parents accessed the system during update implementation.

Solution: Coordinated with exam unit managers to schedule system updates during low-traffic periods. Implemented proper deployment procedures and user communication protocols to balance development velocity with user experience.
